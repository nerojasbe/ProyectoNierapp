Options

Check the "Create backup archive with compression" box to have StuffIt Scheduler use compression when creating the backup archive. This option is on by default.

Check the "Verify the archive..." box to have StuffIt Scheduler verify that the backup archive was created successfully. This option is on by default.

Check the "Perform this task with administrative privileges" box if you wish to backup files belonging to users other than yourself. This option may also be helpful for eliminating some of the "File permissions errors" that you may see listed in the right pane of the "Logs" window for a particular task. This option is off by default.

StuffIt Scheduler also allows you to create your backup archive with encryption. Choose the compression method you want to use, and specify the archive that will be used to encrypt the archive. This password will be used everytime the task you are creating is run.
