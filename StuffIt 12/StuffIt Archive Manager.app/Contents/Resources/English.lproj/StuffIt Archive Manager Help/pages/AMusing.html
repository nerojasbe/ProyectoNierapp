Using Archive Manager

The help topic on using Archive Manager has been broken down into several smaller sub-topics to make it easier to find the information you are looking for.

  • Browsing and Expanding Archives: Provides information on browsing the contents of archives, and selectively extracting items from thhe archive without expanding the entire archive. You can use Archive Manager to browse StuffIt, Zip, TAR, and RAR archives.
  • Creating and Modifying Archives: Provides information about using Archive Manager to create new archives, and about modifying the contents of existing archive files. You can use Archive Manager to modify the contents of StuffIt and Zip archives.
  • Working with Archive Collections: Archive Manager includes powerful tools that make it easy for you to manage the archives you create with StuffIt or receive from other people. Using a "Type Manager" style of interface, similar to iTunes or iPhoto, you can organize your archives into Collections based on a wide range of criteria such as archive name, type, or date of creation.
  • Searching for Archived Items: Provides information about StuffIt's Spotlight plugin, which enables you to search for items stored in StuffIt, Zip, and TAR archives.
  • Index Archives: Provides information about creating and working with StuffIt Index Archives.
  • The StuffIt X File Format: Provides information about the StuffIt X file format.
  • StuffIt Scheduler: Provides information about creaating scheduled archive tasks using the StuffIt X file format.
