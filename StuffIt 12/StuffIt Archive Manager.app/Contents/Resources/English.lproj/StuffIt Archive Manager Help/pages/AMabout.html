About Archive Manager

Using Archive Manager you can browse the contents of StuffIt, Zip, TAR, and RAR archives without first expanding them. Archive Manager enables you to selectively extract only the files you want from a much larger archive.

Archive Manager also allows you to work with most StuffIt and Zip in much the same way as you would work with files and folders in the Finder; using Archive Manager you can add, remove, rename, move, copy, and delete files, as well as create new folders to better organize the data in your archives.

Previously a separate application called Archive Assistant, in StuffIt 12, Archive Manager now has the ability to schedule archive creation. Click here to find out more about using StuffIt Scheduler.

Archive Manager also includes powerful tools to help you work with and manage the archives you create. Much like Smart Playlists in iTunes and Smart Albums in iPhoto, Archive Manager allows you to create custom Collections of archives that allow you to view all of the data associated with a particular project with a single click. Collections are controlled by a set of search criteria that you define. Archive Manager's Collections are tightly integrated with Apple's Spotlight search technology to provide fast and optimized searches that are automatically updated.

Some examples:

  • Create a Collection that contains just the StuffIt archives in your Documents folder that have been either created or changed within the last week.
  • Create a Collection that contains any archives located in your Downloads folder.

Archive Manager Collections can collect more than just archives! An Archive Manager Collection can contain any type of file. Using this feature, you can use Archive Manager as a project manager application to keep track of all the files and folders associated with a particular task and easily archive them.
