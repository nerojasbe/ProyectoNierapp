Archive Search

In previous versions of the StuffIt Deluxe suite, a separate application, Archive Search, was included that allowed users to search for items stored in StuffIt, Zip, or TAR archives. This functionality is now provided by way of a Spotlight plugin. When StuffIt Deluxe is installed, Spotlight is able to extend its indexing capabilities to include the names of all the files located within StuffIt, Zip, and Tar archives. When you use Spotlight to run a search, if a matching file is located within an archive, that archive is returned as part of Spotlight's search results.

Note: Encrypted archives are excluded from Spotlight's indexing.
