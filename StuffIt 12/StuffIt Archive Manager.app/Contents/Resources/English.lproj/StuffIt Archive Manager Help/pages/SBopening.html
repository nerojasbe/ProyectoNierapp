Launching StuffIt Scheduler

When you first launch the StuffIt Scheduler application, you'll see an empty list. As you create and schedule backup tasks, they'll show up in this list. You'll be able to see when the task was last run, and when it is scheduled to run again. You can also edit the details of the tasks from this dialog, but more about that later. To get started creating your first backup task, click the "New Task" button on the StuffIt Scheduler toolbar.
