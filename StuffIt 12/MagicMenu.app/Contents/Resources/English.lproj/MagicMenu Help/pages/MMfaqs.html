Frequently Asked Questions

Q: The Magic Menu is loaded in my menu bar, but when I pull it down I don't see any compression commands listed. Why is this?

A: Magic Menu only lists compression commands when you have items in the Finder selected. You must first select a file(s) and/or a folder(s) in the Finder before you use Magic Menu to select a command. Single items can be selected in the Finder by simply clicking them with your mouse. Multiple items can be selected in the Finder by holding down the Command (Apple) key while you click the items with your mouse.


Q: The Magic Menu does not appear in my menu bar or the StuffIt contextual menu does not appear when I Control + click an item in the Finder. How can I get them to appear?

A: Typically these menus should be available after you install StuffIt Deluxe. If one or both of them is not appearing, Log out of your user account and Log back in again. Then go to your StuffIt Deluxe folder and double-click the MagicMenu icon.


Q: I don't want to use Magic Menu. Rather than Quit it everytime I login, can I prevent it from launching?

A: Yes. Go to System Preferences ->Accounts. Select your account and click the "Startup Items" tab. Select the MagicMenu item and click the minus ('-') at the bottom of the window. When you login into your account, MagicMenu will no longer launch, but you will still have access to the StuffIt Contextual Menu.


Q: Who can expand StuffIt X (.sitx) archives?

A: Both Macintosh and Windows users can, provided they have a copy of our free expansion tool, StuffIt Expander. For the most backwards-compatible archive check the "Create Backwards Compatible Archives" option in DropStuff's StuffIt preferences. The most current version of StuffIt Expander can be downloaded from: StuffIt.com

OS Version Recommended Expander Version
Mac OS 8.6 - 9.2.2 (Classic Mac OS) - StuffIt Expander 7.0.3
Mac OS 10.0 (Cheetah), 10.1 (Puma), 10.2 (Jaguar) - StuffIt Expander 8.0.2
Mac OS 10.3 (Panther) - StuffIt Expander 10.0.2
Mac OS 10.4 (Tiger) - StuffIt Expander 12.0
Windows 98SE - StuffIt for Windows 9.0
Windows 2000 or later - StuffIt for Windows 11.0.1

Those users that cannot expand .sitx files for whatever reason should be sent StuffIt (.sit) archives which can be opened by StuffIt Expander 5.0 and later. We recommend StuffIt Expander 5.5 or higher for these archives. StuffIt Expander 5.5 works only on the Classic Mac OS versions, not on OS X. It and many other legacy versions of Aladdin/Allume/Smith Micro software, can be found in our software library archives. At the time of this writing the StuffIt Expander 5.5 can be found here:
